# 전기차 충전소 데이터 시각화 대시보드

이 프로젝트는 전기차 충전소 데이터를 시각화하여 사용자에게 유용한 정보를 제공하는 대시보드를 개발하는 것을 목표로 합니다.  
대시보드는 충전소 위치, 사용 현황, 충전 속도 등 다양한 데이터를 시각적으로 표현하여 사용자 경험을 향상시킵니다.
